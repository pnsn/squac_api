SQUACapi
********

Intro
=====
SQUACapi is a RESTful API to manage metrics for seismic station data.
This docuement covers the use of the API and not how to set up an instance
of SQUACapi.


`SQUACapi GitHub <http://github.com/pnsn/squacapi>`_
